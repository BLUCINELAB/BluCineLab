SPATIAL OBJECT

A large-format installation conceived as an immediate visual gesture: compact, memorable and designed to hold presence within a public environment.

ROLE

INSTALLATION CONCEPT

Scale logic, spatial gesture and overall presence.

FIELD

TEMPORARY STRUCTURE

Installation design for high-visibility environments.

STATUS

CONFIDENTIAL

The project remains under partial embargo and is shown here in abstract form.

CONTEXT

THE PROJECT

The intervention works through scale, clarity and recognisability. It functions both as structure and as image.

METHOD

THE APPROACH

The installation is developed as one strong readable move: visible from distance, formally compact and able to hold attention in a dense context.

PROJECT SNAPSHOT

Type

Installation

Focus

Scale, iconicity, spatial impact

Output

Immediate recognisability and architectural presence
