INSTALLATION / SPATIAL STRUCTURE

COSMOPROF

A large golden cube installation conceived as a strong fair presence: 3 metres by 3 metres, spatially immediate and visually memorable.

ROLE

Installation concept / spatial structure / visual presence

FORMAT

3 × 3 metre golden cube

FIELD

Installation / fair design / spatial communication

CONTEXT

THE PROJECT

The project works through scale, clarity and immediate recognisability. The object must function both as structure and as image.

APPROACH

THE METHOD

The installation is designed as a single strong gesture: compact, legible, present from distance, and capable of holding attention within a crowded fair environment.
