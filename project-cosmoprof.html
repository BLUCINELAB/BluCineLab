SPATIAL INSTALLATION

Confidential large-format installation conceived as a strong fair presence: spatially immediate, visually memorable and built around impact at scale.

ROLE

Installation concept / spatial structure / visual presence

FORMAT

Large-scale installation object

FIELD

Installation / fair design / spatial communication

CONTEXT

THE PROJECT

The project works through scale, clarity and immediate recognisability. The object must function both as structure and as image.

APPROACH

THE METHOD

The installation is designed as a single strong gesture: compact, legible, present from distance and capable of holding attention in a crowded environment.
